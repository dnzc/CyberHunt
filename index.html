How do the challenges work?

Challenge Answers

The solution to each challenge is a link to the next challenge, contained in a string called a flag. The flag always starts with the string flag{ and ends with } i.e. it is wrapped in flag{}.

The url of the next challenge is always in the format https://dnzc.github.io/Cyber-Hunt/challenges/{{FLAG}}/challenge.html

For example, if the solution to a challenge was:

flag{hello_world-1337}

Then the url of the next challenge would be:

https://dnzc.github.io/Cyber-Hunt/challenges/hello_world-1337/challenge.html

Challenge Briefings

Each challenge comes with a briefing that outlines the goal for that challenge, as well as any challenge files. The briefings page can be accessed >>here<< , go there to get started - no login / signup is required, just start solving :)

Other Information

Please don't use any automated scanning tools such as sqlmap / dirbuster - these are not needed at all.

If you visit a page and it returns a 404 error, then that page really doesn't exist, don't bother trying to uncover anything.

The challenges are sequential and are designed to generally increase in difficulty. There are many miscellaneous topics covered that require a good general knowledge about computers and a lot of creativity. Good luck!

If enough people are interested, I am happy to provide personal help.

I'm stuck. How can I get help?

Keep trying! These challenges are not meant to be trivial, and if you persist, the eureka moment will come, I promise.
